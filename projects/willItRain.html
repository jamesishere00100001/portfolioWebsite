Concept

Will It Rain is a lightweight weather app using live weather data from an OpenWeather API, performing a network call to retieve the weather JSON data and manipulating to display current weather condtions and update the app background to suit. Apple SF symbols are used to display current weather conditions in tandem with JSON derived condition details displayed in text format.

Functionality allows for user based location search in addition to typed city search.

Design

The basic layout of the app was built during a tutorial and then updated and customised to allow for additional functionality and more granular data presentation.

Native symbols and locally stored images are used to provide the user with current weather conditions either in their current location via the CoreLocation intergation or within a location entered manually via the device soft keyboard.

Challenges

Decoding and handeling of the 'detailed' weather conditions associated with the JSON return and intepretation and presentation of the information both in text and by means of the background image update was the primary challenge.

Weather condition data was filtered through a detailed switch statement to extract the catergorised weather conditions and then passed to the relavant methods to update the UI respectively.
